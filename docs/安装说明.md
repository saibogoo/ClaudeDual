# ClaudeDual 安装说明

ClaudeDual 是一个用于管理 Claude Desktop 开发者模式和第三方模型代理配置的 macOS 工具。

## 安装

1. 打开 `ClaudeDual.dmg`
2. 将 `ClaudeDual.app` 拖到“应用程序”目录
3. 双击运行 `ClaudeDual`

## 如果 macOS 提示无法打开

如果系统提示“无法验证开发者”或阻止打开，可以执行下面的命令信任本应用：

```bash
sudo xattr -r -d com.apple.quarantine /Applications/ClaudeDual.app
```

然后再次打开 `ClaudeDual.app`。

如果你还没有把 App 拖到“应用程序”目录，而是在下载目录中运行，则使用：

```bash
sudo xattr -r -d com.apple.quarantine ~/Downloads/ClaudeDual.app
```

执行后可以右键点击 `ClaudeDual.app`，选择“打开”。

## 首次使用

1. 打开 ClaudeDual
2. 在“状态”页点击“一键开启开发者模式”
3. 选择或编辑配置
4. 点击当前激活配置右侧的“启动”

ClaudeDual 会自动创建 `~/Library/Application Support/ClaudeDual-3p` 数据目录，并写入基础配置。
